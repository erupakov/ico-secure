Bitcoin addresses offline generator
===================================

To generate addresses just run genkeys.sh, it will create 2 json files:
1. secretlist.json that keep both address and privkey values
2. publiclist.json that keep only addresses and should be uploaded to the online server

Feel free to play with values in script to match expectations, it's not so complex.
